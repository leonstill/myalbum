# 我的相册web服务

- 浏览指定目录的所有视频。
- 标记图片、视频文件，不修改原文件名；(*)

## 版本说明
v0.01                   建立目录结构，完成测试代码中stream服务及视频seek功能。
v0.02                   视频服务器启动时自动生成缩略图。(*)

## 目录结构
```
config\                 全局配置
stream\                 流媒体服务程序
www\                    前端页面服务程序
test\                   测试代码存放目录
videoserver\            视频服务器
lib\                    通用库目录
README.md               本说明文件
```
